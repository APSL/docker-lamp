========================
Docker LAMP base project
========================

Get started
===========

1. clone::

    git clone https://github.com/APSL/docker-lamp.git

2. build::

    cd docker-lamp
    docker build -t lamp .

3. run::

    bin/lamp start

OR use docker hub: 

1. pull: 
    docker pull apsl/lamp

2. run: 
    docker run  apsl/lamp
